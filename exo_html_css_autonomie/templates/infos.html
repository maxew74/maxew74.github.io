Informations complémentaires

Les langages à utiliser sont HTML et CSS uniquement.

Il s'agit d'un exercice purement technique. Le contenu n'a pas d'importance. Les textes utilisés sont fictifs ("Lorem ipsum..."). Vous pouvez les conserver tels quels ou bien les remplacer par des textes de votre choix.

Site "one-page"

Le site est réalisé en une seule page, scrollable. Un seul fichier HTML est donc attendu.

Pour réaliser la navigation entre les différentes parties, utilisez des liens internes (ex : "#monlien"), ainsi que la pseudo-classe CSS ":target" pour définir le style des parties de contenu à afficher.

Astuce

Pour positionner le sommaire (ensemble des liens de navigation), donner la valeur sticky à la propriété position de l'élément <nav>.
