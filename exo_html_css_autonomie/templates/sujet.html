Sujet

Ceci est un exercice de perfectionnement technique en intégration web, à réaliser en autonomie.

Objectif principal :

Il s'agit de reproduire le plus fidèlement possible le site web présenté en démonstration (vidéo et screenshots) ci-après :

Vous devrez donc réaliser :

  • La structure du contenu (HTML)
  • La mise en page et les effets graphiques (CSS)
  • La navigation en mode "site one-page" (HTML + CSS)
  • Les effets visuels d'apparition des éléments (CSS)
  • Les effets visuels au survol des liens (HTML + CSS)

Objectif secondaire :

Imaginer les adaptations et réaliser les media-queries nécessaires pour rendre le site responsive.
