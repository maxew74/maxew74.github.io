Sélecteurs avancés :

* { /* Sélecteur "universel", cible tous les éléments */ }

selecteur_1, selecteur_2 { /* Propriétés des éléments ciblés par 'selecteur_1' ET 'selecteur_2' */ }

selecteur_1 selecteur_2 { /* Propriétés des éléments ciblés par 'selecteur_2', ENFANTS des éléments ciblés par 'selecteur_1' */ }

selecteur_1 > selecteur_2 { /* Propriétés des éléments ciblés par 'selecteur_2' dont le parent est ciblé part 'selecteur_1' */ }

selecteur_1 + selecteur_2 { /* Propriétés des éléments ciblés par 'selecteur_2' directement précédés de 'selecteur_1' */ }

selecteur_1 ~ selecteur_2 { /* Propriétés des éléments ciblés par 'selecteur_2' précédés de 'selecteur_1' */ }

D'autres sélecteurs CSS utiles sur : http://www.w3schools.com/cssref/css_selectors.asp
