Durée du cours : 28H (8 séances)

Programme prévisionnel :
  • Séance 1 : Généralités, HTML, CSS (rappels théoriques) | Maxime Dressenetto
  • Séances 2 et 3 : Travail sur les maquettes | Isabelle Biamonti + Maxime Dressenetto
  • Séances 4 à 7 : Réalisation du sujet, intégration | Isabelle Biamonti + Maxime Dressenetto
  • Séance 8 : Présentation et rendu des travaux, Évaluation | Isabelle Biamonti + Maxime Dressenetto

Objectifs pédagogiques :

A l'issue de ce cours, je dois savoir :
  • Mettre en place un environnement de travail HTML/CSS "from scratch"
  • Concevoir et réaliser un site web statique
  • Préparer les éléments (fichiers, médias...) et les organiser de façon pertinente.
  • Écrire du code HTML et CSS (respect des bonnes pratiques)
  • Utiliser les outils de déboggage
  • Rechercher les références appropriées pour compléter mes connaissances
  • Mettre en ligne les fichiers (par ftp)

Sujet : Trello "Intégration web"

Sujet : une expérience immersive dans l'univers des fables de La Fontaine.

Le site devra être responsive, orienté "mobile-firts" et réalisé dans les versions :

  • mobile
  • desktop

Il devra comporter minimum 3 pages.

Imaginer et maquetter la navigation entre les pages, ainsi que les fonctionnalités qui pourraient s'avérer pertinentes en réponse au sujet.

> Le rendu devra être uploadé sur votre espace web.
> Il devra aussi être optimisé, 'zippé', puis transmis aux formateurs via Trello + hébergement en ligne.

Qualités attendues pendant le cours : Créativité, Rigueur/Organisation, Proactivité.
