Pseudo-classes :

Les pseudo-classe CSS permettent de cibler des éléments du DOM dans un état spécifique.

selecteur:pseudo-class { /* propriétés CSS */ }

:hover { /* Propriétés des balises de type 'balise1', lorsqu'on les survole */ }

:first-child { /* cible un élément qui est le premier noeud enfant de son parent */ }

:last-child { /* cible un élément qui est le dernier noeud enfant de son parent */ }

:nth-child(n) { /* élément en position 'n' */ }

:nth-child(odd) { /* éléments en position impaire */ }

:nth-child(even) { /* éléments en position paire */ }

:focus { /* cible un l'élément sur lequel le focus */ }

:link { /* lien non visité */ }

:active { /* lien actif */ }

:visited { /* lien visité */ }

:target { /* cible un élément dont l'id correspond à l'ancre présente dans l'URL */ }

D'autres pseudo-classes : https://www.w3schools.com/css/css_pseudo_classes.asp
