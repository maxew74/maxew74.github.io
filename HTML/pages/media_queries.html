La balise "viewport" :

<head>
<meta name="viewport" content="user-scalable=no, initial-scale=1, maximum-scale=1, width=device-width, height=device-height">
</head>

Pour en savoir plus : https://developer.mozilla.org/fr/docs/Mozilla/Mobile/Balise_meta_viewport

Media queries :

La définition de media queries permet de définir des propriétés CSS en fonction des caractéristiques du device de l'utilisateur.

@media not/only mediatype and (media feature) {

propriétés CSS;

}

mediatype : all, screen, print type de média

media feature :
  • max-width largeur maximum du viewport
  • max-height hauteur maximum du viewport
  • orientation landscape ou portrait
  • prefers-color-scheme: dark, light mode dark vs light
  • ...

Plus de détails sur : http://www.w3schools.com/cssref/css3_pr_mediaquery.asp
