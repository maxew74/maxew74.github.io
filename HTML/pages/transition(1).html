Transition :

La propriété transition définit les propriétés CSS qui pourront être animées lors du changement d'état d'un élément (ex : survol).

div{

...
width : 100px; largeur de l'élément

transition : propriété durée (s, ms); définition d'une propriété à animer
transition-delay : 0 (s, ms); délai avant la transition

}

div:hover{

...
width : 200px; largeur de l'élément au survol

}

D'autres propriétés sur : http://www.w3schools.com/cssref/css3_pr_transition.asp
