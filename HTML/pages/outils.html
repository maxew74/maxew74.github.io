Consultation de pages :

Pour consulter une page HTML, on a besoin d'un navigateur web (Firefox, Chrome, Safari, Opéra, Edge, Internet Explorer...).

Écriture du code :

Éditeurs de texte :

Pour écrire le code, on utilise un éditeur de texte (Notepad++, Brackets (IOs), Sublime Text, Atom, Visual Sudio Code...).

Préprocesseurs HTML / CSS :

Les préprocesseurs sont des outils qui permettent d'optimiser la production de code HTML ou CSS. Ils offrent un gain de temps et davantage de souplesse dans l'écriture du code.

Afficher la source de la page :

On peut généralement afficher le code source d'une page web à partir d'un clic droit
