Pseudo-élements :

Les pseudo-élements CSS permettent "d'ajouter virtuellement" des classe ou du contenu au HTML.

selecteur::pseudo-element { /* propriétés CSS */ }

selecteur::before { /* insère un élément AVANT les éléments correspondant à 'selecteur', grâce à la propriété 'content' */ }

selecteur::after { /* insère un élément APRES les éléments correspondant à 'selecteur', grâce à la propriété 'content' */ }

selecteur::first-letter { /* cible la premère lettre */ }

selecteur::first-line { /* cible la premère ligne */ }

selecteur::marker { /* cible les puces de liste */ }

selecteur::selection { /* cible les éléments sélectionnés par l'utilisateur */ }

D'autres pseudo-classes et pseudo-elements CSS : https://www.w3schools.com/css/css_pseudo_elements.asp
