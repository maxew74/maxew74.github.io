Durée du cours : 28H (8 séances)

Programme prévisionnel :
  • Apports thérorique : Généralités, HTML, CSS
  • Exercices
  • Mise en pratique : Réalisation du sujet, intégration
  • Présentation et rendu des travaux, Évaluation

Objectifs pédagogiques :

A l'issue de ce cours, je dois savoir :
  • Mettre en place un environnement de travail HTML/CSS "from scratch"
  • Concevoir et réaliser un site web statique
  • Préparer les éléments (fichiers, médias...) et les organiser de façon pertinente.
  • Écrire du code HTML et CSS (respect des bonnes pratiques)
  • Utiliser les outils de déboggage
  • Rechercher les références appropriées pour compléter mes connaissances
  • Mettre en ligne les fichiers (par ftp)

Sujet :

Réaliser un jeu vidéo 2D.

Le site devra être responsive, orienté "mobile-first" et réalisé dans les versions :

  • mobile
  • desktop

Il devra comporter minimum 3 pages.

  • Minium 3 écrans(un écran d’introduction + un écran de gameplay + un écran de fin)
  • Au moins 1 affichage de données (jauge / vie / points...)
  • Minimum 3 éléments animés
  • Une navigation pour passer entre les écrans
  • Vous devrez imaginer et maquetter le concept, le scénario(histoire/univers), ainsi que les fonctionnalités
> Le livrable devra être optimisé, 'zippé', nommé "Nom_prenom_titre.zip", puis transmis aux formateurs via Trello

Qualités attendues pendant le cours : Créativité, Rigueur/Organisation, Attitude proactive.
