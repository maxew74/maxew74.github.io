Filtres :

Pour appliquer un filtre, on définit ainsi la propriété filter : type( valeur );

Il est possible d'appliquer plusieurs filtre à un même élément. Il faut pour cela séparer les différentes déclaration par un espace.

		grayscale *  (0-1) | % * niveaux de gris
		blur * (px) * flou
		saturate *  (0-1) | % * saturation
		sepia *  (0-1)  | % * effet sepia
		hue-rotate *  0 - 360 deg * teinte
		invert * (0-1) * inversion de couleurs
		brightness * (0-1) * luminosité
		contrast * (0-...) * constrastes
		opacity *  (0-1) | % * opacité

Modes de fusion :

		mix-blend-mode *  color, color-burn, multiply, darken, lighten, ... * mode de fusion

Clipping :

		clip-path *  circle(r), ellipse(l h at x y), polygon(...), path('svg path') * découpage de l'élément
