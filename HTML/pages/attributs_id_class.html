Pourquoi faire ?

  • Identifier les différents éléments
  • Appliquer des styles génériques ou spécifiques aux éléments

Attribut ID :

Permet d'identifier un élément en particulier. Doit être unique dans une même page.

<balise id="identifiant">Contenu</balise>

Attribut CLASS :

Permet d'attribuer une classe de style à plusieurs éléments.

<div class="classe">Contenu</div>
<div class="classe">Contenu</div>
<div class="classe">Contenu</div>
