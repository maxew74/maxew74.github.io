Définition de styles CSS :

/* 'ceci est un commentaire' */


balise {
propriété_1:valeur_1;
propriété_2:valeur_2;
}


#id {
propriété1:valeur1;
propriété2:valeur2;
}


.class {
propriété_1:valeur_1;
propriété_2:valeur_2;
}


/* Utilisation de "variables" CSS */

:root { /* pseudo classe qui représente la racine du document */
/* déclaration d'une variable 'globale' */
--ma-variable: black;
}

.exemple{
color: var(--ma-variable);
}

NB : Certaines propriétés ou valeurs CSS3 n'étant pas toujours prises en compte par les navigateurs, il est parfois nécessaire d'y ajouter un préfixe :

  • -webkit- (pour Chrome, Safari, ...)
  • -moz- (pour Firefox)
  • -o- (pour Opera)
  • -ms- (pour Internet Explorer)
