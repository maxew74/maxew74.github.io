Propriétés d'une animation :

		animation-name * nom * nom de l'animation
		animation-duration * s * duréee de l'animation
		animation-direction * normal | reverse | alternate | alternate-reverse * direction de l'animation
		animation-iteration-count * nombre | infinite * répétitions de l'animation
		animation-play-state * running | pause * état de l'animation

D'autres propriétés sur : http://www.w3schools.com/cssref/css3_pr_animation.asp

Définitions des images-clés :

@keyframes une_animation {

from {
propriétés initiales
}
to {
propriétés finales
}

}
@keyframes une_autre_animation {

0% {
propriétés au début de l'animation
}
50% {
propriétés à la moitié de l'animation
}
100% {
propriétés à la fin de l'animation
}

}
