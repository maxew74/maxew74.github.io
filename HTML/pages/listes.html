Listes non ordonnées :

<ul> : balise ouvrante liste

<li> item 1 </li> : élément de liste 1

<li> item 2 </li> : élément de liste 2

<li>...</li> : ...

</ul> : balise fermante liste

  • item 1
  • item 2
  • ...

Listes ordonnées :

<ol> : balise ouvrante liste

<li> item 1 </li> : élément de liste 1

<li> item 2 </li> : élément de liste 2

<li>...</li> : ...

</ol> : balise fermante liste

  1. item 1
  2. item 2
  3. ...
